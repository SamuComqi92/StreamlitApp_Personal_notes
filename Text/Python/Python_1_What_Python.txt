### What is Python?

**Python** is a high-level programming language created by *Guido van Rossum* during the latest ‘80s and named after the BBC show “Monty Python’s Flying Circus”.

It has a very simple syntax, and it can be used for many tasks (for example, *Data Visualization, Data Analysis, Machine Learning*, etc.).

- ***Interpreted***: no need to compile before executing your program.
- ***Interactive***: you can write a command and interact with it.
- ***Object-Oriented***: encloses code within object

.
